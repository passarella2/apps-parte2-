Sobre o AppHub Pro

O AppHub Pro é um projeto **open-source** criado para demonstrar o poder do **HTML5**, **CSS3** (com variáveis) e **JavaScript moderno**. É uma vitrine de utilitários simples, mas eficientes, desenvolvida por **passarella2**.

Este projeto é distribuído sob a **Licença MIT**. Sua contribuição é bem-vinda!
